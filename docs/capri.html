];

target.hair === "brown"; //true

reloadPage([clearCache])

Reload page
Provides option to preserve local cache of resources (images, scripts, styles). This is achieved by adding/altering query argument t. This means the current scroll offset would be lost. This issue is resolved by introducing a URI fragment that contains the vertical offset in pixels #y=123.
Where possible (browser support) the scroll offset fragment is automatically removed upon loading page.
Cache is cleared (when specified) by reloading window.
